# ao-pages
full document template for Apple Pages

# Install
- clone or download _this_ project
- _dblClick_ on `ao-pages.template` & select _'Add to Template Chooser'_

- In pages, ``` Menu >File >New >``` 'My Templates' section - now has template _'ao-pages'_ for selection
